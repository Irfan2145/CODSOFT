# Simple Calculator

A basic calculator built using HTML, CSS, and JavaScript.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Percentage
- Clear Button
- Delete Button
- Responsive Layout

## Technologies Used

- HTML
- CSS
- JavaScript

## Project Structure

```text
Calculator/
│
├── index.html
├── style.css
├── script.js
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open index.html in your browser.
